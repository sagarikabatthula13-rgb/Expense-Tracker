# Expense-Tracker
Expense Tracker

Project Description

This is a simple Python Expense Tracker application. The program allows users to enter multiple expenses, calculates the total amount spent, and displays the final total.

Features

- Add multiple expenses
- Automatic total calculation
- Easy-to-use command-line interface
- Uses Python loops and arithmetic operations

Technologies Used

- Python 3

How to Run

1. Save the file as "expense_tracker.py"
2. Open a terminal in the project folder
3. Run the command:

python expense_tracker.py

4. Enter expense amounts when prompted
5. Type "done" to finish and view the total spent

Example

Enter expense amount: 100

Enter expense amount: 50

Enter expense amount: 20

Enter expense amount: done

Total Spent: 170.0
